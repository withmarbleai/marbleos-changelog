# Talk Mode

Have a back-and-forth conversation with Marble in the new Talk mode. Click the “Talk Beta” button on the upper left corner to access it. You can always come back to the older Marble interface if you like.

## Visual understanding

Marble now visually understands your website, so it can generate graphics that align with your style.

Click the pencil icon and then add your website with the "Add Website" button. To show Marble how your site looks, you will need to add your website again even if you've done it before.
